# Leaf It Alone Web (3D Edition)

낙엽을 치우고 정원을 관리하는 3D 시뮬레이션 게임입니다. 도구를 사용하여 낙엽을 모으고, 봉투에 담아 배수구로 배달하세요!

## 실행 방법 (Run Steps)
1. 의존성 설치: `npm install`
2. 개발 서버 실행: `npm run dev`
3. 브라우저에서 `http://localhost:3000` 접속

## 조작 방법 (Controls)
- **이동 (Move)**: `W`, `A`, `S`, `D`
- **점프 / 봉투 던지기 (Jump / Throw Bag)**: `Space`
- **상호작용 (Interact)**: `E` (게이트 해금 / 알림 닫기)
- **도움말 (Help)**: `P` (도움말 패널 토글)
- **도구 선택 (Select Tools)**: 
  - `1`: 손 (Manual Pick)
  - `2`: 갈퀴 (Rake) - *넓은 범위 수집*
  - `3`: 송풍기 (Blower) - *강력한 바람 발사*
  - `4`: **진공 청소기 (Vacuum)** - *100개 수집 시 자동 봉투 생성*
- **UI**:
  - `I`: 인벤토리 (Inventory)
  - `U`: 상점 (Shop)

## 스테이지 가이드 (Stage Guide)
**총 5개의 스테이지**로 구성되어 있으며, 각 단계마다 새로운 방해 요소가 등장합니다.

| 스테이지 | 이름 | 특징 및 방해 요소 |
|:---:|:---:|:---|
| **1** | **뒷마당 (Backyard)** | 평화로운 튜토리얼 구역. 1,200개의 낙엽을 모으세요. |
| **2** | **확장 구역 (Expansion)** | 본격적인 시작. 낙엽을 모아 봉투에 담고 배수구(Drain)에 파세요. |
| **3** | **두더지의 습격 (Mole Attack)** | **방해꾼 두더지**가 등장합니다! 8초마다 튀어나와 낙엽을 흩뿌립니다. |
| **4** | **바람의 언덕 (Windy Hill)** | **강풍(Air Vent)**과 더 강력해진 두더지가 등장합니다. |
| **5** | **폭풍의 눈 (Storm Area)** | **최종 난이도**. 거대 **토네이도**와 **천둥 번개(Thunder)**가 몰아칩니다. <br> - **20m 방벽**: 낙엽 유실 방지를 위한 고공 투명벽 설치 <br> - **180도 회전된 작은 휴지통**: 봉투 넣기가 매우 어렵습니다. |

## 주요 기능 (Key Features)
- **리얼타임 물리 엔진**: 수천 개의 낙엽이 바람, 도구, 충돌에 반응합니다.
- **자동 수집 배수구**: 봉투를 배수구에 던지면 즉시 돈으로 환전됩니다.
- **다이내믹 로딩**: 스테이지 이동 시 각 구역에 맞는 고유 로딩 화면이 제공됩니다.
- **진행형 난이도**: 스테이지가 지날수록 두더지 주기 단축, 바람 세기 증가 등 난이도가 상승합니다.

## 개발 참고 사항
- **기술 스택**: Next.js, React Three Fiber, Cannon.js (Physics), Zustand
- **최적화**: InstancedMesh를 사용하여 수천 개의 낙엽을 효율적으로 렌더링합니다.
- **확장성**: `scenes.ts` 설정 파일을 통해 스테이지 구성(목표, 배치, 방해 요소)을 쉽게 수정할 수 있습니다.
